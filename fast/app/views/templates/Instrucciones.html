INSTRUCCIONES DE MANEJO

Modo Manual

  • Utiliza las flechas direccionales para controlar el movimiento del vehículo
  • ↑ - Avanzar hacia adelante
  • ↓ - Retroceder
  • ← - Girar a la izquierda
  • → - Girar a la derecha

Modo Automático

  • El vehículo se conducirá de forma autónoma
  • El sistema detectará automáticamente obstáculos y señalización
  • Puedes tomar el control manual en cualquier momento

Funciones Adicionales

  • Botón "TOMAR FOTO" - Captura una imagen del entorno actual
  • Botón "IDENTIFICAR OBSTÁCULO" - Analiza y muestra información sobre obstáculos detectados

Inicio de Operación

  • Presiona "INICIAR STREAM" para activar la cámara
  • Selecciona el modo de operación (Manual/Automático)
  • Verifica que todos los sistemas estén funcionando correctamente
  • Comienza la operación de manera gradual
